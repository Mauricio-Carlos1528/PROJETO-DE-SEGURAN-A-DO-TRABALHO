📌 COMO USAR A APLICAÇÃO
═══════════════════════════════════════════════════════════════════════════════

1️⃣  ABRIR A APLICAÇÃO
    • Navegue até a pasta do projeto
    • Abra o arquivo: index.html
    • Qualquer navegador moderno funciona (Chrome, Firefox, Safari, Edge)

2️⃣  PREENCHER O FORMULÁRIO
    • Digite seu NOME COMPLETO
    • Use o controle deslizante para avaliar:
      └─ Disposição Física (1=Muito cansado, 5=Excelente)
      └─ Estado Mental (1=Muito estressado, 5=Focado/Tranquilo)
    • Indique se sente DOR ou DESCONFORTO
    • Confirme se está APTO para trabalhar
    • Adicione OBSERVAÇÕES (opcional)
    • Clique em "ENVIAR AVALIAÇÃO DIÁRIA"

3️⃣  VISUALIZAR HISTÓRICO
    • Clique na aba "Histórico de Testes"
    • Use a busca para encontrar registros de um colaborador
    • Veja todas as avaliações passadas em ordem cronológica

4️⃣  NOTAS IMPORTANTES
    ✓ Os dados são salvos LOCALMENTE no seu navegador
    ✓ Não é necessário conectar-se à internet
    ✓ Se limpar o cache/cookies, os dados podem ser deletados
    ✓ Faça backup dos dados regularmente

❓ DÚVIDAS FREQUENTES
═══════════════════════════════════════════════════════════════════════════════

P: Como faço para abrir a aplicação?
R: Abra o arquivo "index.html" em seu navegador web (Chrome, Firefox, etc).

P: Preciso de internet para usar?
R: Não! A aplicação funciona 100% offline no seu navegador.

P: Onde meus dados são salvos?
R: No banco de dados local do navegador (IndexedDB), no seu próprio computador.

P: E se eu limpar o cache do navegador?
R: Os dados podem ser deletados. Recomendamos fazer backups regularmente.

P: Posso usar em diferentes computadores?
R: Não. Os dados estão salvos no navegador do computador onde você acessa.

P: Como faço backup dos dados?
R: Visite a aba de Histórico e copie os dados. (Futuro: função de export)

P: A aplicação funciona em celular?
R: Sim! Design responsivo - funciona em smartphones e tablets.

P: Qual navegador usar?
R: Chrome, Firefox, Safari ou Edge (versões recentes).
