Como Começar

Passo 1: Abrir a Aplicação

Abra o arquivo index.html em seu navegador web (Chrome, Firefox, Safari ou Edge)

Passo 2: Preencher o Formulário

  • ✓ Digite seu nome completo
  • ✓ Avalie sua disposição física (1-5)
  • ✓ Avalie seu estado mental (1-5)
  • ✓ Indique presença de dor
  • ✓ Confirme aptidão para trabalhar
  • ✓ Adicione observações (opcional)

Passo 3: Enviar Avaliação

Clique no botão "Enviar Avaliação Diária" para salvar seus dados

Passo 4: Visualizar Histórico

Clique na aba "Histórico de Testes" para ver todos os registros e filtrar por colaborador

Descrição dos Arquivos

Arquivo principal - Estrutura HTML da aplicação

js/config.js

Configurações centralizadas (DB, mensagens, cores)

js/utils.js

Funções utilitárias reutilizáveis (formatação, validação, etc)

js/script.js

Lógica principal da aplicação (formulário, abas, histórico)

db/database.js

Gerenciamento do banco de dados IndexedDB

css/styles.css

Estilos customizados que complementam Tailwind CSS

Banco de Dados

A aplicação usa IndexedDB - um banco de dados local no navegador.

✅ Vantagens

  • • Funciona offline
  • • Sem necessidade de servidor
  • • Dados salvos localmente

💾 Dados Armazenados

  • • Nome do colaborador
  • • Data e hora
  • • Avaliações (físico/mental)
  • • Observações

⚠️ Importante

  • • Limpar cache = perder dados
  • • Faça backup regularmente
  • • Use navegador confiável
